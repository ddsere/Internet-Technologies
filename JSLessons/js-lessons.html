JavaScript Lessons

1. DOM Selectors

DOM selectors are methods used to select HTML elements from the document so you can manipulate them with JavaScript.
Details: You can select elements by their ID, class, tag name, or CSS selector.
Examples: 

// Select by ID
const title = document.getElementById('main-title');

// Select by class
const items = document.getElementsByClassName('list-item');

// Select by tag name
const paragraphs = document.getElementsByTagName('p');

// Select using CSS selector
const firstButton = document.querySelector('.btn');
 
// Select all elements matching a CSS selector
const allButtons = document.querySelectorAll('.btn');
// Returns a NodeList of all elements with class 'btn'

2. Modifying Element Attributes

You can change the attributes of HTML elements (like src, href, class) using JavaScript.
Details: Use setAttribute to add or change an attribute, and removeAttribute to delete one.
Examples: 

// Change the image source
const img = document.getElementById('logo');
img.setAttribute('src', 'new-logo.png');

// Add a class to an element
img.setAttribute('class', 'highlighted');

// Remove an attribute
img.removeAttribute('alt');

3. Modifying Element Content

JavaScript allows you to change the text or HTML inside an element using properties like innerHTML and textContent.
Details: innerHTML can set HTML markup, while textContent sets plain text.
Other useful properties: innerText sets or gets the visible text of an element, and value is used for form elements like input, textarea, and select.
Examples: 

// Change the text inside a paragraph
const para = document.querySelector('p');
para.textContent = 'This is new text!';

// Insert HTML into a div
const box = document.getElementById('box');
box.innerHTML = '<strong>Bold text!</strong>';

// Change the visible text of an element
const heading = document.getElementById('main-heading');
heading.innerText = 'Welcome to JavaScript!';

// Change the value of an input field
const input = document.getElementById('username');
input.value = 'JohnDoe';

4. JavaScript Integration with HTML

JavaScript can be integrated with HTML to make web pages interactive.
Details: You can use event listeners to respond to user actions like clicks, mouse movements, or keyboard input.
Examples: 

// Add a click event to a button
const btn = document.getElementById('myBtn');
btn.addEventListener('click', function() {
  alert('Button was clicked!');
});

// Change content when input changes
const input = document.getElementById('nameInput');
input.addEventListener('input', function() {
  document.getElementById('output').textContent = input.value;
});
